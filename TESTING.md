# Technical test checklist

Use the HTTPS-hosted page rather than opening `index.html` directly from the computer.

## Core questionnaire

- Complete all six ratings.
- Complete all fifteen comparisons.
- Review, edit and submit.
- Confirm that the weighted score is produced.

## Accessibility support

- Standard and large text.
- Standard scale and smiley landmarks.
- Simpler explanations.
- Large answer and navigation controls.
- Keyboard-only navigation and visible focus.
- Built-in audio guidance.
- External screen reader.
- Voice recognition and confirmation.
- Save, close, reopen and resume.
- Gaze camera setup, calibration, dwell proposal, separate confirmation and camera stop.

## Recommended environments

- Chrome or Microsoft Edge for gaze and voice testing.
- NVDA with Chrome or Edge for Windows screen-reader testing.
- VoiceOver with Safari for a separate semantic-accessibility check.

Record each item as Pass, Partial, Fail or Not supported.
